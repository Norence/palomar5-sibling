# The Metal-Poor Sibling of Palomar 5
Discovery and characterization of the Zhujiang River stream in the southern Galactic halo.

## Overview
This repository contains the analysis code for the discovery of a metal-poor stellar stream that appears to be a sibling structure to the well-known Palomar 5 stream.

## Data
Data files are stored in the `data/` directory and are not tracked in git due to size. 
Contact the authors for data access.
